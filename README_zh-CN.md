# Pin Items 模组

![预览](./PinItems/release_extra/preview.png)

这是一个简洁的功能性模组，让你可以在背包中**固定物品（Pin）**，在使用 **“全部存入（Store All）”** 时，它们不会被自动存进仓库。

## 功能特性

- 将任意物品标记为**已固定（Pinned）**，在你点击 `Store All` 时，这些物品会留在背包里。
- 再也不用在每次清理背包后，从仓库里把常用装备一件件翻出来。
- 已固定的物品仍然可以单独移动到仓库，只是移入时会自动取消固定状态。

## 使用方法

- 打开你的背包界面。
- 选中你想一直带在身上的物品。
- 在物品菜单中选择 `Pin` 选项，将其固定。
- 之后当你点击 `Store All` 时，已固定物品会留在背包，其余物品会照常存入仓库。
- 如果你手动把某个已固定物品移动到仓库，它会在移动时自动**取消固定**。

让背包管理更轻松，再也不用和仓库反复拉扯。
