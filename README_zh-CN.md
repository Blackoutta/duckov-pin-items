# Pin Items 模组

![预览](./PinItems/release_extra/preview.png)

这是一个简洁的功能性模组，让你可以按**物品类型**来固定（Pin）装备，在使用 **“全部存入（Store All）”** 时，这些类型的物品会拒绝被自动塞进仓库。

## 功能特性

- 给喜欢用的子弹、工具或药品固定一次，之后所有同样的物品都会在 `Store All` 时自动留在背包中。
- 保持常用道具组合，不用再在每次清包后去仓库翻出来。
- 固定状态不会因为手动整理而消失，随便拖动也不会误解除。

## 使用方法

- 打开背包或仓库界面。
- 选中你想长期保留在身上的道具。
- 在物品菜单中点击显眼的 `固定` 按钮切换固定状态。
- 之后当你点击 `Store All` 时，所有已固定的物品都会留在背包，其余照常入库。
- 需要时你仍可手动把固定物品拖进仓库，固定状态将保持不变。

## Pin 与锁定（`L` 键）的区别

- Pin 针对的是“物品类型”而不是槽位。
- Pin 不会影响排序，它只是在执行 Store All 时阻止系统把固定的道具塞进仓库里。
- Pin 按钮位于物品菜单中，位置显眼，一眼就能找到。
- Pin不会与锁定冲突，两者可以一起使用！


让背包管理更轻松，再也不用和仓库反复拉扯。
